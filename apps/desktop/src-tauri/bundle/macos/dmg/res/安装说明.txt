# 安装说明

如果打开应用时提示「已损坏，无法打开」，请按以下步骤操作：

## 方法一：右键打开（推荐）

1. 将「Grok Build」拖到「应用程序」文件夹
2. 打开「应用程序」文件夹
3. 右键（或按住Control键点击）「Grok Build」
4. 选择「打开」
5. 在弹出的对话框中点击「打开」

## 方法二：使用终端命令

1. 将「Grok Build」拖到「应用程序」文件夹
2. 打开「终端」（启动台 -> 其他 -> 终端）
3. 复制粘贴以下命令并回车：

```bash
xattr -cr /Applications/Grok\ Build.app
```

4. 现在可以正常打开应用了

## 方法三：在系统设置中允许

1. 打开「系统设置」（或「系统偏好设置」）
2. 进入「隐私与安全性」
3. 滚动到「安全性」部分
4. 找到提示「Grok Build 被阻止...」的消息
5. 点击「仍要打开」

---

注意：本应用是开源项目，未经苹果官方公证。以上方法可以安全地打开应用。
